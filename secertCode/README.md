### From a String to a Secert Code - Create a function that takes: 

1. One **Lowercase String** as an argument. 
1. Show **Argument String** in a **Secert Coded** version in the console.

Note : For converting string into a secret code **Replace All** 'a's with 4, 'e's with 3, 'i's with 1, 'o's with 0, and 's' s with 5..

> strToSecertCode("thursday") ➞ "thur5d4y"

> strToSecertCode("javascript is fun") ➞ "j4v45cr1pt 15 fun"

> strToSecertCode("lets go india") ➞ "l3t5 g0 1nd14"
