### Reverse an Array - Create a function that takes: 

1. **One Array** as an argument. 
1. Show **Reversed Array** in the **console**.

> ([1, 2, 4]) ➞ [4, 2, 1]

> ([10, 15, 20, 25]) ➞ [25, 20, 15, 10]

> (['a', 'b', 'c', 'd', 'e']) ➞ ['e', 'd', 'c', 'b', 'a']
