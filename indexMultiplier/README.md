### Index Multiplier - Create a function that takes: 

1. An **Integer Array** as an argument. 
1. **Multiply Each** array **Item** with it's **Index Number**, and show the **Sum** of **Entire Multiplied Array** in the console.

> arrindexMultiplier([26, 7, 1996, -11, -23]) ➞ 3874

> arrindexMultiplier([30, 4, 1999, -15, -55]) ➞ 3737

> arrindexMultiplier([55, 18, -19, 15, -7]) ➞ -3
