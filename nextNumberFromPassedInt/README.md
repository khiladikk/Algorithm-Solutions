### Next Number From The Passed Integer - Create a Function that takes:

1. **One Integer** (Number) as an argument.

2. **Increments** the integer (Number) by **+1** and show the Result in the console.

> Examples : nextNumber(10) ➞ 11

> nextNumber(-4) ➞ -3

> nextNumber(21) ➞ 22
