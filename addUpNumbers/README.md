### Add up the Numbers from a Single Number - Create a function that takes: 

1. One **Number** as an argument. 
1. **Add Up** all the numbers from **1 to the Passed** number.

> addUpNumbers(30) ➞ 465

> addUpNumbers(05) ➞ 15 

> addUpNumbers(19) ➞ 190

Note : Pass positive number between 1 to 1000.
